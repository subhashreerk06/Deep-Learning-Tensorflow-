# Deep Learning with TensorFlow

A personal learning repository for practicing and understanding deep learning concepts using **TensorFlow** and **Keras**.

## About

This repository contains beginner-to-intermediate deep learning programs that I wrote while learning TensorFlow. The aim of this repo is to document my progress, organize my practice work, and build a clear foundation in neural networks and applied deep learning.

## Files in this Repository

- **Heart_Disease.py**  
  A deep learning model for heart disease prediction using tabular data.

- **Image_CNN.py**  
  A Convolutional Neural Network (CNN) implementation for image-based learning tasks.

- **Image_Classification.py**  
  A neural network model for image classification.

## Goals of this Repository

- Learn the fundamentals of deep learning
- Practice TensorFlow and Keras workflows
- Build intuition for model training, validation, and evaluation
- Maintain a structured record of my learning progress

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- scikit-learn

## Notes

This is a **learning repository**, so the code is primarily intended for study, experimentation, and improvement over time.
